Promise in JavaScript : A Promise in JavaScript is an object that represents the result of an asynchronous operation
                        e.g : You place an order → Pending
                              Food delivered → Resolved
                              Order cancelled → Rejected


3 States of a Promise
    Pending → initial state (waiting)
    Resolved (Fulfilled) → success
    Rejected → failure

Syntax : 

const promise = new Promise((resolve, reject) => {
  // async work
});

- A Promise is a way to handle future results of async operations in a clean and structured way.

----------------------------------------------------------------------------------------------------------------------------------------------------
Synchronous Operations : one by one execution (line-by-line). Execution is blocking — next line won’t run until current line finishes.

Asynchronous Operations : does NOT wait, runs in background . Tasks can run independently, without blocking the main flow.
Common Async Methods : 1.  setTimeout : Runs a function once after a delay. It does not block execution (asynchronous). 
                                    Use cases:
                                        -Delay execution
                                        -Show notifications after some time
                                        -API retry after delay
                       2.  setInterval : Runs a function repeatedly after a fixed interval. Keeps running forever until stopped.
                                     Use cases:
                                        -Clock updates ⏰
                                        -Auto-refresh data
                                        -Polling APIs
                       3.  Promises : A Promise is used to handle asynchronous operations and represents a value that may be available now, later, or never






fetch() : built-in JavaScript function used to make HTTP requests (get data from APIs)

Promise Chaining : Promise chaining is a pattern where you attach multiple .then() calls to a Promise so that the output of one step becomes the input of the next.Promise chaining means executing asynchronous tasks in sequence, passing results from one .then() to the next.
eg : Promise.resolve(5)
  .then(num => num * 2)
  .then(num => num + 3)
  .then(result => console.log(result));


How It Works : Each .then():

-Receives result from previous step
-Returns a new value (or Promise)
-Passes it to next .then()
